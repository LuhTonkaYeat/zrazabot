# 🍽 Zraza Bot

Личный tg-бот для чата с кентами. Считает съеденные зразы (локальный рофл), добавляет случайные гарниры и редкие фразы.

## Команды

- `/zraza` — съесть порцию зраз (от 1 до 10). Случайный гарнир и редкие фразы
- `/zrazastat` — топ-5 легенд столешницы по количеству зраз

## Технологии

- Go 1.23
- SQLite
- gopkg.in/telebot.v3
- modernc.org/sqlite

## Деплой

Бот работает на [BotHost](https://bothost.ru) в Docker контейнере с постоянным хранилищем `/app/data`.

## Переменные окружения

- `BOT_TOKEN` — токен бота от @BotFather

## Особенности

- ⏰ Кулдаун 1 час между командами
- 🍝 4 вида гарниров
- 💩 10% шанс получить редкую фразу с 💩
- 📊 Рейтинг с именами пользователей
